# Housing Price Prediction System 🏠

A complete end-to-end housing price prediction system with React frontend and FastAPI backend.

## Features

### Backend (FastAPI)
- ✅ Housing price prediction using ML models
- ✅ Indian Rupee (₹) currency formatting
- ✅ Single and batch predictions
- ✅ Market analysis and statistics
- ✅ Model performance metrics
- ✅ CORS enabled for frontend

### Frontend (React)
- ✅ Property value estimator with form validation
- ✅ Interactive price charts and visualizations
- ✅ Prediction history with export
- ✅ Property comparison view
- ✅ Market analysis dashboard
- ✅ What-if analysis tool
- ✅ Responsive design with Tailwind CSS
- ✅ Error boundaries and loading states
