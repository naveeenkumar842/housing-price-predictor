# 🏠 Housing Price Prediction System

A complete end-to-end housing price prediction system with React frontend and FastAPI backend, featuring Indian Rupee (₹) currency support.

## ✨ Features

### Backend (FastAPI)
- 🔮 **Price Prediction**: Single and batch predictions using ML models
- 📊 **Market Analysis**: Comprehensive statistics and visualizations
- 💰 **Indian Rupee Support**: All prices formatted in ₹ with proper Indian numbering
- 📈 **Model Metrics**: R² score, RMSE, MAE, and feature importance
- 🏥 **Health Check**: Simple endpoint to verify API status

### Frontend (React)
- 📝 **Property Estimator**: Easy-to-use form with validation
- 📊 **Interactive Charts**: Visual price comparisons
- 💰 **Price Breakdown**: See how features contribute to price
- 📜 **Prediction History**: Track all your predictions
- 🎯 **Market Analysis**: Interactive dashboard with filters
- 🔄 **What-If Analysis**: Adjust features and see price impact
- 📤 **Export Data**: CSV and PDF export options
